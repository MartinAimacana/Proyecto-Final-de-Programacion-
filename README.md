# Sistema de Órdenes de Trabajo en C

Este proyecto corresponde al desarrollo de un sistema de órdenes de trabajo realizado en lenguaje C como parte de la asignatura Programación I.

El programa funciona mediante un menú en consola y permite gestionar órdenes de manera básica, aplicando los conceptos aprendidos durante el segundo semestre.

---

## Descripción general

El sistema permite registrar, almacenar y administrar órdenes de trabajo.  
Cada orden contiene información del cliente, equipo, tipo de trabajo, costo base, horas trabajadas y el costo total, el cual se calcula automáticamente según una tarifa fija por hora.

El objetivo del proyecto es poner en práctica el uso de estructuras, funciones, arreglos y archivos en lenguaje C.

---

## Funcionalidades

El sistema permite realizar las siguientes acciones:

- Registrar nuevas órdenes de trabajo.
- Listar todas las órdenes registradas.
- Buscar órdenes por código o por nombre del cliente.
- Actualizar información de una orden existente.
- Eliminar órdenes con confirmación.
- Guardar los datos en un archivo CSV.
- Cargar automáticamente la información al iniciar el programa.
- Salir del sistema con guardado automático.
